Welcome to edward.js

A simple presentation tool
using HTML5, CSS and jQuery

<<Use your arrow keys to continue >>

Slides are written entirely in HTML...

...and spiced up with CSS and JS

  • Your content is plain HTML, easy to edit and share
  • Presentation "themes" or "skins" are just CSS stylesheets
  • The edward.js engine manages transitions
  • Using javascript we can extend these and implement new ones

Slides are written entirely in HTML...

...and spiced up with CSS and JS

  • Your create the slideshow providing a container
  • Every child element of it became slides
  • Adding data elements you specify transitions and time delay
  • Just look at the source code of this presentation ;)

Callbacks

Events and hooks in slides

  • Your can have callbacks for onShow and onHide
  • Execute function calls when a slide gets in or out

Design with freedom

You can add links and img's too. Even videos.

  • You can use styles and Javascript. Richer slides!
  • Links in the content : www.google.com. Links between slides too
  • Code block :

    <?php
require('module.php');
echo('edward rules!');
exit();

If you know CSS and JS, you can add your own extensions

  • Use CSS to implement themes and styles
  • API and callbacks(work in progress)
