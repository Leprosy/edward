Slides are written entirely in HTML...

...and spiced up with CSS and JS

  • Your content is plain text, easy to edit and share
  • Presentation styles are implemented with CSS themes
  • The edward.js engine manages transitions
  • Using jQuery we can extend animations and other logic

Slides are written entirely in HTML...

...and spiced up with CSS and JS

  • Your can have steps into the slides too
  • Only add a steps="true" attribute to your lists
  • The edward.js engine manages transitions
  • And do the animations

Callbacks

  • Execute function calls when a slide gets in or out
